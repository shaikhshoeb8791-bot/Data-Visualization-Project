# 📊 Data Visualization Project

## 📌 Overview

This project was completed as part of my Data Analytics Internship at Decode Labs. The objective was to transform raw data into meaningful visualizations that clearly communicate insights and support data-driven decision-making.

## 🎯 Project Objective

* Analyze the dataset using Python.
* Create effective visualizations to identify trends, distributions, and relationships.
* Present insights in a clear and easy-to-understand format.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## 📈 Visualizations Created

* Bar Chart
* Line Chart
* Pie Chart
* Histogram
* Scatter Plot

## 🔍 Key Insights

* Compared product sales using bar charts.
* Analyzed sales trends over time with a line chart.
* Visualized payment method distribution using a pie chart.
* Examined the distribution of total order values with a histogram.
* Explored the relationship between quantity and total price using a scatter plot.
